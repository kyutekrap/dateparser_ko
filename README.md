# dateparser_ko
한글에서 날짜 추출하기<br>
최신 버전: 0.0.1


### 되는 거
- 상반기, 하반기
- 1~4분기
- 올해, 작년, 재작년
- 4개월 전, 4개월치


### 안되는 거 !important
- 주 단위 (이번주, 지난주)
- 일 단위 (오늘, 내일)
- 미래 날짜 (다음 해, 다음 주)
- 긴 표현 (최근 4년 동안)


### 기타 주의사항
- 순서가 유의미함 (2024년부터 3년치 - OK, 2024년까지 3년치 - NO)
- N개의 날짜가 포함된 문구는 테스트가 상대적으로 미흡했음.
- 띄어쓰기, 오타 고려 안함.
